# JKLMbot
un bot imbattable et insoupçonnable a JKLM BombParty 
comment le lancer: 
assurez vous d'avoir python et edge sur votre ordinateur 
installez les librairies python pyautogui unicodedata keyboard selenium 
dans votre terminal lancez a la racine du projet py JKLMbot
rentrez le lien de votre partie 
si edge s'ouvre c'est que vous avez réussi!
